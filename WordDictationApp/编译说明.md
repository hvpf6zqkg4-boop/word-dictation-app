# 词语听写小助手 - 编译说明

## 环境要求

### 1. 开发环境
- **Android Studio**: 最新版本（推荐 Arctic Fox 或更高）
- **JDK**: 11 或更高版本
- **Android SDK**: API 24 (Android 7.0) 或更高
- **Gradle**: 7.0+（Android Studio 自带）

### 2. 系统要求
- **操作系统**: Windows 10/11, macOS 10.14+, Ubuntu 18.04+
- **内存**: 至少 8GB RAM
- **磁盘空间**: 至少 10GB 可用空间

## 编译步骤

### 方法一：使用 Android Studio（推荐）

1. **打开项目**
   - 启动 Android Studio
   - 选择 "Open" 或 "Open an Existing Project"
   - 选择 `WordDictationApp` 文件夹

2. **同步项目**
   - Android Studio 会自动检测并同步 Gradle 依赖
   - 如果提示 "Gradle sync failed"，请检查网络连接并重试

3. **配置 SDK**
   - 确保已安装以下 SDK 组件：
     - Android SDK Platform 34
     - Android SDK Build-Tools 34.0.0
     - Android Emulator（如需测试）
   - 在 File → Project Structure 中检查 SDK 配置

4. **编译 APK**
   - 选择 Build → Build Bundle(s) / APK(s) → Build APK(s)
   - 或点击工具栏的 "Run" 按钮（绿色三角形）
   - 编译完成后，APK 文件位于：
     ```
     app/build/outputs/apk/debug/app-debug.apk
     ```

### 方法二：使用命令行

1. **安装必要工具**
   ```bash
   # 确保已安装 JDK 11+
   java -version
   
   # 确保已安装 Android SDK
   # 设置环境变量（Windows）
   set ANDROID_HOME=C:\Users\用户名\AppData\Local\Android\Sdk
   set PATH=%PATH%;%ANDROID_HOME%\tools;%ANDROID_HOME%\platform-tools
   
   # Linux/macOS
   export ANDROID_HOME=~/Android/Sdk
   export PATH=$PATH:$ANDROID_HOME/tools:$ANDROID_HOME/platform-tools
   ```

2. **编译 APK**
   ```bash
   # 进入项目目录
   cd WordDictationApp
   
   # 清理项目
   ./gradlew clean
   
   # 编译调试版 APK
   ./gradlew assembleDebug
   
   # 编译发布版 APK（需要签名）
   ./gradlew assembleRelease
   ```

3. **找到 APK 文件**
   - 调试版：`app/build/outputs/apk/debug/app-debug.apk`
   - 发布版：`app/build/outputs/apk/release/app-release-unsigned.apk`

## 签名 APK（发布版）

### 1. 生成密钥库
```bash
keytool -genkey -v -keystore my-release-key.jks \
  -keyalg RSA -keysize 2048 -validity 10000 \
  -alias my-alias
```

### 2. 配置签名
在 `app/build.gradle` 中添加：
```gradle
android {
    signingConfigs {
        release {
            storeFile file("my-release-key.jks")
            storePassword "your_password"
            keyAlias "my-alias"
            keyPassword "your_password"
        }
    }
    buildTypes {
        release {
            signingConfig signingConfigs.release
            minifyEnabled false
            proguardFiles getDefaultProguardFile('proguard-android-optimize.txt'), 'proguard-rules.pro'
        }
    }
}
```

### 3. 编译签名 APK
```bash
./gradlew assembleRelease
```

## 常见问题解决

### 1. Gradle 同步失败
**问题**: Could not resolve all dependencies
**解决**:
- 检查网络连接
- 修改 `build.gradle` 中的仓库地址：
  ```gradle
  repositories {
      google()
      mavenCentral()
      // 添加国内镜像
      maven { url 'https://maven.aliyun.com/repository/google' }
      maven { url 'https://maven.aliyun.com/repository/public' }
  }
  ```

### 2. 权限问题
**问题**: Camera permission denied
**解决**:
- 确保在 AndroidManifest.xml 中声明了权限
- 在代码中动态请求权限
- 在手机设置中手动授予权限

### 3. TTS 初始化失败
**问题**: TTS language not supported
**解决**:
- 确保设备支持中文语音
- 首次使用需要下载语音包
- 检查网络连接

### 4. OCR 识别失败
**问题**: No text recognized
**解决**:
- 确保图片清晰
- 文字方向正确
- 光线充足
