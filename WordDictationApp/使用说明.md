# 词语听写小助手 - 使用说明

## 应用简介

**词语听写小助手**是一款专为小朋友设计的 Android 应用，可以帮助家长和老师进行词语听写练习。通过拍照或选择图片，应用会自动识别图片中的中文词语，并按设定的时间间隔逐个朗读出来。

## 主要功能

1. **图片识别**: 拍照或从相册选择包含词语的图片
2. **文字识别**: 自动识别图片中的中文词语（完全离线）
3. **语音朗读**: 使用系统语音引擎朗读词语
4. **间隔控制**: 可调节朗读间隔时间（5-120秒）
5. **播放控制**: 支持开始、暂停、继续、停止

## 系统要求

- **Android 版本**: 7.0 (API 24) 或更高
- **存储空间**: 至少 50MB 可用空间
- **相机**: 需要后置摄像头（用于拍照）
- **语音**: 需要中文语音包（首次使用会自动下载）

## 首次使用指南

### 1. 安装应用
- 下载并安装 `app-debug.apk`
- 如果提示"未知来源"，请先启用"允许安装未知来源应用"

### 2. 权限设置
首次打开应用时，需要授予以下权限：
- **相机权限**: 用于拍照功能
- **存储权限**: 用于选择相册图片
- **语音权限**: 用于语音合成

### 3. 下载语音包（首次使用）
- 首次使用 TTS 功能时，系统会提示下载中文语音包
- 请确保网络连接正常
- 下载完成后即可正常使用

## 详细使用步骤

### 第一步：选择图片
1. **拍照方式**
   - 点击主界面"拍照"按钮
   - 对准包含词语的纸张或屏幕
   - 点击拍照按钮拍摄
   - 照片会自动返回主界面

2. **相册方式**
   - 点击主界面"从相册选择"按钮
   - 从相册中选择包含词语的图片
   - 图片会自动加载到应用中

### 第二步：识别词语
1. 选择图片后，应用会自动开始识别
2. 识别过程中会显示"正在识别图片中的词语…"
3. 识别完成后会显示"识别成功！找到X个词语"
4. 识别的词语会显示在列表中

### 第三步：设置朗读间隔
1. 使用滑块调节朗读间隔时间
2. 范围：5秒 - 120秒
3. 默认：30秒
4. 实时显示当前设置的时间

### 第四步：开始朗读
1. 点击"开始"按钮开始朗读
2. 当前朗读的词语会高亮显示
3. 显示朗读进度（第X个/共X个）
4. 按设定的时间间隔自动朗读下一个词语

### 第五步：控制朗读
1. **暂停**: 点击"暂停"按钮暂停朗读
2. **继续**: 点击"继续"按钮继续朗读
3. **停止**: 点击"停止"按钮停止朗读
4. **重新开始**: 停止后点击"开始"重新开始

## 使用技巧

### 1. 图片拍摄技巧
- **光线充足**: 确保拍摄环境光线充足
- **对焦清晰**: 点击屏幕对焦，确保文字清晰
- **角度垂直**: 尽量垂直拍摄，避免透视变形
- **背景简洁**: 选择简洁的背景，避免干扰

### 2. 词语识别技巧
- **字体清晰**: 使用清晰易读的字体
- **字号适中**: 字号不宜过小
- **排列整齐**: 词语排列整齐，避免重叠
- **间距合适**: 词语间保持适当间距

### 3. 朗读设置技巧
- **初学者**: 建议设置 30-60 秒间隔
- **进阶者**: 可设置 15-30 秒间隔
- **挑战者**: 可设置 5-15 秒间隔
- **根据年龄**: 年龄越小，间隔越长

## 常见问题解答

### Q1: 应用无法识别图片中的文字
**可能原因**:
1. 图片不清晰
2. 光线太暗
3. 文字太小
4. 字体特殊

**解决方法**:
1. 重新拍摄清晰图片
2. 改善拍摄光线
3. 调整文字大小
4. 使用标准字体

### Q2: 语音朗读没有声音
**可能原因**:
1. 音量太小
2. 未下载语音包
3. 设备不支持中文语音
4. TTS 引擎故障

**解决方法**:
1. 调大设备音量
2. 下载中文语音包
3. 检查设备语音设置
4. 重启应用或设备

### Q3: 应用闪退或卡顿
**可能原因**:
1. 内存不足
2. 设备性能较低
3. 应用版本不兼容
4. 系统问题

**解决方法**:
1. 关闭其他应用
2. 清理设备存储
3. 更新应用版本
4. 重启设备

### Q4: 无法保存或选择图片
**可能原因**:
1. 存储权限未授予
2. 存储空间不足
3. 文件系统问题
4. 应用权限被限制

**解决方法**:
1. 授予存储权限
2. 清理存储空间
3. 检查文件系统
4. 重置应用权限

### Q5: 相机无法使用
**可能原因**:
1. 相机权限未授予
2. 相机被其他应用占用
3. 相机硬件故障
4. 系统相机服务问题

**解决方法**:
1. 授予相机权限
2. 关闭其他相机应用
3. 检查相机硬件
4. 重启设备
